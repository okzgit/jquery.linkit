#LinkIt jQuery Plugin#
**LinkIt** is a simple plug in to attach links to elements.

##Versions##
0.3.0

##Usages##
    $('span').linkIt({
			href: 'http://www.test.com',
			text: 'Click Me',
			target: '_blank'
		});


##Vendors##
jQuery - [http://jquery.com](http://jquery.com)
